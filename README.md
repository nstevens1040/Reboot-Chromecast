# Reboot-ChromeCast
Windows PowerShell script that will reboot Chromecast devices on my local network.  
```ps1

NAME
    Reboot-Chromecast
    
SYNTAX
    Reboot-Chromecast [-DeviceName] {Nick's TV | Google TV}  [<CommonParameters>]
    
    
PARAMETERS
    -DeviceName <string>
        
        Required?                    true
        Position?                    0
        Accept pipeline input?       false
        Parameter set name           (All)
        Aliases                      None
        Dynamic?                     false
```
